# 🧲 MagnetiCrea — Flutter App

App per piastrelle magnetiche con guida animata passo-passo.

## Come avviare

```bash
flutter pub get
flutter run
```

## Struttura

- `lib/models/` — Modelli dati (TileType, Construction, BuildStep)
- `lib/data/` — Catalogo piastrelle e costruzioni
- `lib/providers/` — State management con Riverpod
- `lib/screens/` — Schermate principali
- `lib/widgets/` — Widget riusabili (TilePainter)
